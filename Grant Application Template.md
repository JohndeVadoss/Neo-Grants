## Project Roadmap

Please structure the development roadmap into multiple milestones. Since the milestones will be included in the grant contract, we recommend the team provide as much detail as possible. When each milestone is delivered, we will refer to the commitment concomitant contract to ensure that everything has been delivered as per expectations.

Please prioritize these important criteria as the team prepares the roadmap and milestones:

* Focus: Project brings value to the Neo ecosystem.
* Project Timeframe: up to 6 months is ideal.
* Engineering Excellence: in the quality of the software deliverables.
* Developer Guidance: Documentation and/or tutorials for the deliverables.

For each milestone, please provide detail for the following sections:

* **Features** 

  Please include a specific feature/capability list which will be delivered for each milestone, and also provide enough details for subsequent product acceptance review.

* **Verifiable deliverable** 

  Please highlight the iteration plan and corresponding deliverables which should be beta version products can be tested/verified, and also indicate the relevant documentations and/or tutorials.

* **Duration**

  Please estimate the duration for each milestone. 

* **Funding**

  Please include  the total budget plan for each milestone. This may be broken down into the number of days along with labor cost per day. An indication of the number of full-time employees working on each milestone will be highly appreciated.
